STEP 1: CREATE FIGMA FILE
--------------------------
1. Open Figma (desktop app or web: figma.com)
2. Click "New Design File"
3. Rename file to: "Sabine Dashboard v1.0"

STEP 2: IMPORT SVG MOCKUPS
--------------------------
METHOD A - Drag & Drop (Easiest):
   1. Open the "svgs/" folder in File Explorer/Finder
   2. Select mockup-01-overview-desktop.svg
   3. Drag it into your Figma canvas
   4. Done! ✓

METHOD B - File Menu:
   1. In Figma: File → Place Image/Video...
   2. Navigate to the "svgs/" folder
   3. Select mockup-01-overview-desktop.svg
   4. Click anywhere on canvas to place
   5. Done! ✓
